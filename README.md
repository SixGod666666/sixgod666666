# 天气预报软件说明书 #


 ## 项目介绍 ##


> 摘要：本设计使用Android设计技术开发了一种运用在Android系统上的手机天气预报软件，通过该软件可选择地区来获得天气，温度等信息。
> 功能介绍：
1. 遍历全国省市县数据
2. 显示天气信息
3. 手动切换城市
4. 后台自动更新天气
